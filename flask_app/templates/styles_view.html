Generating Art using DCGAN Architecture

This simple interface displays the art styles present in Kaggle's "Painters by Numbers" dataset. The purpose of this project was to explore the DCGAN architecture's ability to generate artwork with given a dataset comprised of paintings labeled by a specific art style. Neural networks leverage the similarities present in a set of images to make classifications. Here the user can interact with the full dataset, and compose their own by selecting paintings through their own definition of what makes images similar.
